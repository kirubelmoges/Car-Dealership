Car Dealership Platform

 Overview

**Best Cars** is a full-stack car dealership web application that connects customers with trusted dealerships across North America. Users can browse dealerships, read customer reviews with sentiment analysis, register for accounts, and post their own reviews.

Live Demo

 **Live Site:** [https://car-dealership-frontend.onrender.com](https://car-dealership-frontend.onrender.com)  
 **API Endpoint:** [https://car-dealership-backend.onrender.com](https://car-dealership-backend.onrender.com)

 Features

 User Features
- **User Authentication** - Register, login, and logout functionality
- **Dealership Directory** - Browse 500+ verified dealerships
- **State Filtering** - Find dealerships by state
- **Customer Reviews** - Read and write dealership reviews
- **Sentiment Analysis** - Automatic sentiment detection (positive/negative/neutral)
- **Responsive Design** - Works on desktop, tablet, and mobile

 Technical Features
-  **Docker Support** - Containerized for easy deployment
-  **RESTful API** - Clean API architecture
-  **PostgreSQL Database** - Production-ready database
-  **Tailwind CSS** - Modern, beautiful UI
-  **Render.com Ready** - One-click deployment configuration

 Tech Stack

 Frontend

**React 18** for UI Framework 
**Tailwind CSS** for Styling & Design 
**React Router DOM** for Navigation 
**Fetch API** for HTTP Requests 

Backend

**Django 4.2** for Web Framework 
**Django REST Framework** for API Development 
**PostgreSQL** for Production Database 
**SQLite** for Development Database 
**Gunicorn** for WSGI Server 

DevOps & Tools

**Docker** for Containerization 
**Render.com** for Cloud Deployment 
**Git** for Version Control
